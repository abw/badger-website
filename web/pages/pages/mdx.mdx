# MDX Pages

MDX pages (with a `.mdx` file extension) can contain any combination of
Markdown markup and React code.

See the <Link href="https://mdxjs.com/" text="MDX website" targetBlank/> for
further information about MDX.

<Example
  Component={MdxFileComp}
  code={MdxFileCode}
  language="mdx"
  expand
/>

Code blocks are rendered using the <Link to="/code/source" code="Source"/>
component.  The language can be specified as the first argument immediately
after the opening <code>```</code> marker.

Additional properties can be added after that, e.g. to set `caption` or
`highlightLines` as shown in these examples.  This requires the
`rehype-mdx-code-props` module which should be added to the `rehypePlugins`
configuration option for the `mdx` plugin in your `vite.config.js` file.  See
the "Configure Vite Plugins" section in the <Link
to="/getting-started/installation" text="Installation"/> page for an example.
