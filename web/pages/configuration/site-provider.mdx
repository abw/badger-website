# `SiteProvider`

<Snippet
  file="Provider.jsx"
  expand
>
  The `SiteProvider` component is the main entry point for the
  web site.  This will usually be called from the main `App.jsx`
  file.  It takes a number of configuration options.
</Snippet>

### Site Customisation

There are various additional properties that allow you to customise the way
the site is displayed.

<PropsTable
  items={[
    [
      'linkUp',
      'true',
      <>Boolean flag to add a link up to the top of the page in each heading - see <Link to="/page/heading" code="Heading"/></>
    ],
    [
      'linkUpIcon',
      'arrow-up',
      <>Icon displayed in the link up to the top of the page - see <Link to="/page/heading" code="Heading"/></>
    ],
    [
      'prevNext',
      'true',
      <>Boolean flag to generate "Previous Page" and "Next Page" links at the bottom of each page</>
    ],
    [
      'prevNextArrows',
      'false',
      <>Boolean flag to generate arrow icons to the previous page and next page at the top of each page</>
    ],
    [
      'sidebarToc',
      'true',
      <>Boolean flag to add the table of contents for the current page in the sidebar</>
    ],
    ['themeStorageKey', 'theme', 'Local storage key for dark/light theme'],
    ['Router', 'PageRouter', 'Site router for pages'],
    ['*', '', 'Any other properties you want to add to the site provider'],
  ]}
/>
