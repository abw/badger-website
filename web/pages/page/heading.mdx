# Heading

The `Heading` component can be used to render a heading block as an `h2`
element.

It accepts a `text` property for the title text (or `title` as an alias for
`text`), or a `code` property for text that should be displayed in a
monospaced font.

It also inserts a heading entry in the table of contents which will be
displayed in the sidebar and any <PageTocLink/> included in the page.

If the `linkUp` configuration option is `true` (which it is by default) then
a link will be generated on the right side which navigates to the top of the
page.  This is displayed as an icon determined by the `linkUpIcon`
configuration option (`arrow-up` by default).  See the <SiteProviderLink/>
page for further details on setting these options.
