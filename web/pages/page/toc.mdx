# PageToc

The `PageToc` component can be used to render a table of contents for the page.
It will contain links to any <HeadingLink/> and <SubHeadingLink/> elements in
the page.

When using MDX the <HeadingLink/> and <SubHeadingLink/> components are
inserted for level two (`##`) and level three (`###`) headings.
